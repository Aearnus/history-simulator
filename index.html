About

Hiya!

This is a little website called History Simulator. It was written in Ruby by Aearnus and TristanBomb because we were bored. It's intended to "simulate" history from 2015 on by tracking information about countries and periodically applying events.

We're plenty aware of a lot of inaccuracies within the simulation, and there are a number of countries not represented (usually due to size - a pixel is only so big), but if you have any feedback or want to contribute, head to the GitHub page!
